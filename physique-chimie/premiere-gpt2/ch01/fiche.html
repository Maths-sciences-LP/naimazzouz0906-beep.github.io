Chapitre 1 – Le champ magnétique — Fiche de révision

Première Bac Pro | Physique-Chimie | Groupement 2

À retenir — L'essentiel du chapitre

Le champ magnétique est une grandeur physique vectorielle associée aux aimants et aux courants électriques. Son unité est le tesla (T).

1. Les aimants

Définition Un aimant permanent crée un champ magnétique sans alimentation électrique. Il possède un pôle Nord (N) et un pôle Sud (S).
Propriétés
  • Pôles de même nom : répulsion
  • Pôles de noms différents : attraction
  • Seuls les matériaux ferromagnétiques (fer, acier, nickel, cobalt) sont attirés
Attention Le cuivre, l'aluminium, le zinc et le plastique ne sont pas attirés par un aimant.

2. Lignes de champ magnétique

Propriété
  • Les lignes de champ sortent du pôle Nord et entrent dans le pôle Sud.
  • Là où les lignes sont resserrées, le champ est intense.
  • Le champ est plus fort près des pôles.

4. Champ magnétique d'une bobine

Définition Un électroaimant est une bobine (solénoïde) alimentée par un courant continu. Il crée un champ magnétique similaire à celui d'un aimant, mais contrôlable par le courant.
Proportionnalité : \[ B = k \times I \]

Pour une bobine sans noyau de fer, le champ magnétique \(B\) est proportionnel au courant \(I\).
\(k\) dépend du nombre de spires et de la géométrie.

Règle de la main droite On enroule les doigts de la main droite dans le sens du courant dans les spires : le pouce indique le sens du champ magnétique à l'intérieur de la bobine (vers le pôle Nord).
Méthode Identifier les pôles d'une bobine :
  1. Repérer le sens du courant dans les spires visibles.
  2. Appliquer la règle de la main droite.
  3. Le côté où le champ sort est le pôle Nord.
  4. Le côté où le champ entre est le pôle Sud.

6. Applications professionnelles

Dans les métiers de l'électrotechnique
  • Relais électromagnétique : petite bobine commandant un contacteur de puissance
  • Capteur de position (effet Hall) : mesure le champ d'un aimant solidaire d'une pièce mobile
  • Moteur électrique : interaction champ magnétique / courant = force (force de Laplace)
  • Vanne solénoïde : bobine actionnant un clapet pour ouvrir/fermer un circuit hydraulique
