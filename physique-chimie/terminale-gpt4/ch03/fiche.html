Chapitre 3 – Chaîne de transmission d'informations — Fiche de révision

Terminale Bac Pro | Physique-Chimie | Groupement 4

À retenir — L'essentiel

Toute transmission d'information repose sur une chaîne : modulateur → canal de transmission → démodulateur. Le canal peut être une onde électromagnétique (radio, lumière) ou mécanique (son). La fibre optique est le canal le plus performant pour les longues distances.

1. Schéma général d'une chaîne de transmission

Les 3 éléments
  1. Modulateur (émetteur) : code l'information sur une onde porteuse
  2. Canal de transmission : transporte l'onde porteuse (câble, air, fibre)
  3. Démodulateur (récepteur) : extrait l'information de l'onde reçue

2. Types d'ondes utilisées

Ondes mécaniques
  • Nécessitent un milieu matériel
  • Exemples : son (air, eau, solides)
  • Vitesse dans l'air : 340 m/s
  • Applications : sonar, acoustique
Ondes électromagnétiques
  • Se propagent dans le vide
  • Vitesse dans le vide : \(c = 3 \times 10^8\) m/s
  • Spectre : radio, micro-ondes, IR, visible, UV, X, γ
  • Applications : radio, Wi-Fi, fibre optique

3. La fibre optique

Structure d'une fibre optique
  • Cœur : silice d'indice \(n_1\) (plus élevé) — guide la lumière
  • Gaine optique : silice d'indice \(n_2 < n_1\) — confine la lumière
  • Gaine de protection : polymère — protection mécanique
Réflexion totale interne :

La lumière est guidée par réflexion totale interne quand l'angle d'incidence \(\theta > \theta_c\).

\[ \sin\theta_c = \frac{n_2}{n_1} \]

Plus l'écart \(n_1 - n_2\) est grand, plus l'angle critique est petit → plus facile de guider la lumière.

5. Numérisation d'un signal

Deux étapes de la numérisation
  1. Échantillonnage : mesure de la valeur du signal à intervalles réguliers (fréquence d'échantillonnage ≥ 2 × fréquence max du signal)
  2. Quantification : codage de chaque mesure sur \(n\) bits (2ⁿ niveaux)

Débit binaire : \(D = f_e \times n\) (bits/s)

6. Calculs utiles

Conversions :
  • 1 octet = 8 bits
  • 1 ko = 10³ octets, 1 Mo = 10⁶ octets, 1 Go = 10⁹ octets
  • 1 kbit/s = 10³ bit/s, 1 Mbit/s = 10⁶ bit/s, 1 Gbit/s = 10⁹ bit/s
Temps de transfert : \[ t = \frac{\text{taille du fichier (bits)}}{\text{débit (bit/s)}} \]
