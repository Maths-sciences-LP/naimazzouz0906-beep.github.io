Chapitre 1 – Produire une image en couleur — Fiche de révision

Terminale Bac Pro | Physique-Chimie | Groupement 4

À retenir — L'essentiel

Une image numérique est une grille de pixels. Chaque pixel code sa couleur en RVB (Rouge, Vert, Bleu) sur 3 × 8 bits = 24 bits, soit plus de 16 millions de couleurs possibles.

1. Capteur d'images

Définition Un capteur d'images est une matrice de pixels photosensibles convertissant la lumière en signal électrique. Le signal est numérisé (converti en nombre) pour chaque pixel.
CapteurTechnologieUtilisation
CCDCharge Coupled DeviceAppareils photo pro, astronomie
CMOSComplementary Metal Oxide SemiconductorSmartphones, webcams

2. Synthèse additive RVB

Synthèse additive

Les écrans produisent les couleurs en combinant trois couleurs primaires émises par les sous-pixels :

  • R + V = Jaune
  • R + B = Magenta
  • V + B = Cyan
  • R + V + B (maximum) = Blanc
  • R = V = B = 0 = Noir

3. Codage des couleurs

Codage sur 8 bits par composante :
  • Valeurs possibles : 0 à 255 (2⁸ = 256 niveaux)
  • Image RVB (24 bits) : \(256^3 \approx 16{,}7\) millions de couleurs
  • Niveaux de gris (8 bits) : 256 niveaux, R = V = B
Taille d'une image non compressée : \[ \text{Taille (octets)} = \text{largeur} \times \text{hauteur} \times \text{profondeur (octets/pixel)} \]

RVB : 3 octets/pixel | Niveaux de gris : 1 octet/pixel

4. Résolution et pixellisation

Résolution La résolution exprime le nombre de pixels par unité de longueur : dpi (dots per inch) ou ppi (pixels per inch). 1 pouce = 2,54 cm.
UsageRésolution recommandée
Écran web72–96 dpi
Impression standard150–200 dpi
Impression offset professionnelle300 dpi
Attention Une image à faible résolution agrandie pour l'impression donnera une image pixellisée (floue). On ne peut pas inventer des pixels : il faut la bonne résolution dès la capture.
