Exercices – Statistiques à une variable

Exercice 1

Notes d'une classe (12 élèves) : 8, 12, 14, 10, 16, 12, 9, 14, 12, 18, 11, 13.

1. Tableau effectifs/fréquences.
2. Effectif de la note 12.
3. Fréquence de la note 14.
N=12. Note 12 : n=3. Note 14 : f=2/12≈16,7%.

Exercice 2

Couleurs préférées (40 personnes) : rouge 12, bleu 18, vert 6, jaune 4.

1. Fréquences de chaque couleur.
2. Angle pour le secteur bleu dans un diagramme circulaire.
Bleu : f=18/40=45%. Angle=0,45×360=162°.

Exercice 3

Températures (°C) sur 10 jours : 18, 22, 19, 25, 21, 23, 20, 24, 22, 21.
Représenter ces données dans un diagramme en bâtons.
Axe x : valeurs de température. Axe y : effectifs. Bâton à 22°C : hauteur 2.
