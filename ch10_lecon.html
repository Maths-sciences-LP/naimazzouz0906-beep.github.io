Chapitre 8 – Fonctions de référence

2de Pro MA-MA  |  Algèbre – Analyse

Ce chapitre présente les trois fonctions de référence fondamentales : la fonction carré, la fonction inverse et la fonction racine carrée. Pour chacune, on étudie la définition, le tableau de valeurs, la représentation graphique et le sens de variation.

1. Fonction carré   (f(x) = x^2)

Définition
La fonction carré est définie sur (mathbb{R}) par : [f(x) = x^2] Sa courbe représentative est une parabole, symétrique par rapport à l'axe des ordonnées (axe (y)).

Tableau de valeurs

(x)(-3)(-2)(-1)(0)(1)(2)(3)
(f(x)=x^2)9410149
Variations
  • (f) est décroissante sur (]-infty,;,0])
  • (f) est croissante sur ([0,;,+infty[)
  • (f) admet un minimum égal à (0) en (x = 0)
Attention
((-3)^2 = 9) et non (-9). Le carré d'un nombre négatif est toujours positif. [ f(-x) = (-x)^2 = x^2 = f(x) ] La fonction carré est dite paire : la parabole est symétrique par rapport à l'axe des (y).
Exemple
Pour (x = -2{,}5) : (f(-2{,}5) = (-2{,}5)^2 = 6{,}25)
Pour (x = 1{,}2) : (f(1{,}2) = (1{,}2)^2 = 1{,}44)

2. Fonction inverse   (g(x) = dfrac{1}{x})

Définition
La fonction inverse est définie sur (mathbb{R}^* = mathbb{R}setminus{0}) par : [g(x) = rac{1}{x}] Sa courbe représentative est une hyperbole composée de deux branches, une dans le 1er quadrant et une dans le 3e quadrant.

Tableau de valeurs

(x)(-4)(-2)(-1)(-0{,}5)(0{,}5)(1)(2)(4)
(g(x)= rac{1}{x})(-0{,}25)(-0{,}5)(-1)(-2)(2)(1)(0{,}5)(0{,}25)
Variations
  • (g) est décroissante sur (]-infty,;,0[)
  • (g) est décroissante sur (]0,;,+infty[)
  • (g) n'a pas d'extremum
Attention
(g) est décroissante sur chaque intervalle séparément, mais pas sur (mathbb{R}^*) entier.
En effet : (g(-1) = -1 < g(1) = 1), mais (-1 < 1), ce qui semble contradictoire si on ne fait pas attention au domaine.
Asymptotes
Quand (x o 0^+), (g(x) o +infty).   Quand (x o +infty), (g(x) o 0^+).
Les axes des coordonnées sont des asymptotes de la courbe.
Exemple
(dfrac{1}{0{,}1} = 10)   et   (dfrac{1}{0{,}01} = 100) : plus (x) est proche de (0), plus (g(x)) est grand en valeur absolue.

3. Fonction racine carrée   (h(x) = sqrt{x})

Définition
La fonction racine carrée est définie sur ([0,;,+infty[) par : [h(x) = sqrt{x}] (sqrt{x}) désigne l'unique réel positif ou nul dont le carré vaut (x).

Tableau de valeurs

(x)(0)(1)(2)(4)(9)(16)(25)
(h(x)=sqrt{x})(0)(1)(approx 1{,}41)(2)(3)(4)(5)
Variations
  • (h) est croissante sur ([0,;,+infty[)
  • (h) admet un minimum égal à (0) en (x = 0)
  • Plus (x) augmente, plus (sqrt{x}) augmente, mais de moins en moins vite
Attention
(sqrt{x}) n'est définie que pour (x geq 0). L'expression (sqrt{-4}) n'a pas de sens dans (mathbb{R}).
De plus : (sqrt{x^2} = |x|) (valeur absolue), et non (x). Par exemple : (sqrt{(-3)^2} = sqrt{9} = 3 eq -3).
Méthode – Simplifier une racine carrée
Pour simplifier (sqrt{a cdot b}) avec (a, b geq 0) : (sqrt{a cdot b} = sqrt{a} imes sqrt{b})
Exemple : (sqrt{12} = sqrt{4 imes 3} = sqrt{4} imes sqrt{3} = 2sqrt{3})

5. Lien entre les fonctions

Propriété fondamentale
Pour tout réel (x geq 0) : [(sqrt{x})^2 = x qquad ext{et} qquad sqrt{x^2} = |x|] Autrement dit, (sqrt{phantom{x}}) et le carré sont des opérations inverses l'une de l'autre sur ([0,;,+infty[).
Méthode – Lire les variations sur un graphique
  1. Regarder si la courbe monte (de gauche à droite) → fonction croissante
  2. Regarder si la courbe descend → fonction décroissante
  3. Identifier les extrema : points les plus hauts (max) ou les plus bas (min)
  4. Lire les intervalles sur l'axe des (x)
