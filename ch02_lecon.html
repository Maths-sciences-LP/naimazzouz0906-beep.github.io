Cours – Statistiques à une variable

1. Vocabulaire

Population : ensemble étudié. Individu : élément de la population. Caractère : propriété mesurée. Effectif : nombre d'individus. Fréquence : proportion.

2. Tableau de données

Valeur \(x_i\)Effectif \(n_i\)Fréquence \(f_i\)
Valeur 1\(n_1\)\(f_1 = n_1/N\)
.........
Total\(N\)1 (ou 100%)

3. Fréquence

\[ f_i = \frac{n_i}{N} \quad \text{(ou } f_i \times 100 \text{ pour le \%)} \]

4. Représentations graphiques

  • Diagramme en bâtons : effectifs discrets
  • Histogramme : données groupées en classes
  • Diagramme circulaire : fréquences en secteurs (angle = \(f_i \times 360°\))
